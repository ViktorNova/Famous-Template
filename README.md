# Quick Start
## Dependencies
- [NodeJS](http://nodejs.org/)
- [Bower](http://bower.io/)
- [git](http://git-scm.com/book/en/v2/Getting-Started-Installing-Git)

## Install
1. Clone this repo
2. In your terminal run the following commands at the project's root directory
3. npm install
3. npm start
4. View your app at http://localhost:3000/
